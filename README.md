# Apoio Regional PE/PB

Landing page MVP para orientação inicial e direcionamento responsável de famílias em Pernambuco, João Pessoa e regiões próximas.

## Objetivo

Validar rapidamente um funil simples:

1. Visitante acessa a página
2. Entende a proposta
3. Envia contato pelo WhatsApp
4. Lead é registrado manualmente em uma planilha/CRM

## Estrutura

- `index.html` — página principal
- `styles/style.css` — identidade visual e responsividade
- `scripts/script.js` — envio do formulário para WhatsApp

## Cuidados

- Não prometer resultado
- Não fazer diagnóstico
- Não expor dados sensíveis
- Coletar apenas informações mínimas
- Usar os contatos apenas para orientação inicial

## Próximas melhorias

- Criar página de instituições parceiras
- Configurar domínio próprio
